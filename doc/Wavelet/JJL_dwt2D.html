JJL_dwt2D

PURPOSE

discrete 2-D wavelet transform

SYNOPSIS

function out = JJL_dwt2D(x, J, af)

DESCRIPTION 

 discrete 2-D wavelet transform

 USAGE:
   w = dwt2D(x, stages, af)
 INPUT:
   x - N by M matrix
       1) M, N are both even
       2) min(M,N) >= 2^(J-1)*length(af)
   J - number of stages
   af - analysis filters
 OUPUT:
   w - Matrix of wavelet coefficients
 EXAMPLE:
   [af, sf] = farras;
   x = rand(128,64);
   J = 3;
   w = dwt2D(x,J,af);
   y = idwt2D(w,J,sf);
   err = x - y; 
   max(max(abs(err)))
